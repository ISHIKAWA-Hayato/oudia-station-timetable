# oudia-station-timetable

oud2ファイルから、体裁の整った駅時刻表を簡単に作成できる Web アプリです。

## できること

- oud2 ファイルをアップロードして読み込み
- 駅時刻表の内容を自動で整形
- きれいなレイアウトでプレビュー表示
- 必要に応じて画像やPDFとして出力できるようにする予定

## 使い方

1. Web アプリを開く
2. oud2 ファイルをアップロードする
3. 時刻表のプレビューを確認する
4. 生成結果を保存・共有する

## 想定している特徴

- ブラウザだけで使える
- インストール不要
- 簡単な操作で時刻表を作れる
- 既存の oud2 データを活用しやすい
